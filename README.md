# Filestat утилита

## Описание

Утилита выводит в стандартный поток вывода информацию о количестве файлов каждого типа (обычный файл, директория, сокет и тд).

### Как запускать

1. Сборка

Исполнить:
```
g++ -std=c++17 -o filestat main.cpp
```

2. Запуск

Исполнить команду в нужно директории:
```
./filestat
```
